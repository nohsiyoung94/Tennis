# Task: Implement Premium Tennis Academy Website (Thesis Tennis)

## Objective
Create a modern, premium tennis academy website inspired by `https://4seasonstennis.co.kr/`.
The design should be "Premium", "Sporty", and "Dynamic".

## Design System
- **Colors**:
  - Primary: Deep Forest Green (`#1a472a`)
  - Accent: Tennis Lime (`#d2e603`)
  - Dark: Charcoal Black (`#111111`)
  - Light: Off-White (`#f8f9fa`)
- **Typography**: Inter (Modern Sans-Serif)
- **Vibe**: Professional, Clean, Energetic.
